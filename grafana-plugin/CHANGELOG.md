# Changelog

## 2.2.0

- Added time-series query type for time-series database.
- Added managed systems query type for fast system selection in query variables.
- Added PDC support for Grafana Cloud.
- Added real-time and PDSv2 near-term history queries support for NetView.
- Added Grafana 13 support.

## 2.1.0

- Added real-time and near-term history from PDSv2 queries support.
- Added Situation Events query support.
- Added Take Action support directly from panel.
